# NUMK

To control the switch for the numeric keyboard.

## Syntax

```text
NUMK <Num>
```

## Parameters

- **If Num is 0, the switch is off,and 1 on.**

## Examples

```text
NUMK 1
```

## Remarks

  The existing command of SEND can fulfill the same function,yet not quite precisely.When NUMK
  is on,sending the key once more will reset the switch at off.
