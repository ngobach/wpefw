# HIDE

To hide the process of PECMD.EXE.

## Syntax

```text
HIDE
```

## Parameters

- **No params.**

## Examples

```text
HIDE
```

## Remarks

- This command is designed to hide the process of PECMD.EXE in case it is mistakenly or
  artificially killed by other programs.
- It is not applicable to the command line,but to config file only.
- The command of SHEL should be used behind HOTK and HIDE.
- Only after SHELL is loaded by the command of SHEL is HIDE able to hide the process of
  PECMD.EXE.
