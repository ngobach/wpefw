# RAND

To return a random number.

## Syntax

```text
RAND [VarName]
```

## Parameters

- **VarName**: to store the random number,RANDOM by default.Practically %RANDOM% works without
  using the command of RAND.

## Examples

```text
RAND CurRand
```

## Remarks

  The outcome of the example is likely to be "14935321432143".
