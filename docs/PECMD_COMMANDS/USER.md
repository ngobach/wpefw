# USER

To set the names of user and corporation in "Property" in the right click menu of
"My Computer".

## Syntax

```text
USER <UserName>,<CorpName>
```

## Parameters

- **The params are characters.**

## Examples

```text
USER mdyblog.blog.163.com,无忧启动论坛
```

## Remarks

- If there exists a single-byte English "," ,it can be replaced with a Chinese double-byte
  one or transfered to a variable.This command is only applicable to a config file.
