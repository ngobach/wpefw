# WALL

To set the wallpaper.

## Syntax

```text
WALL <WallpaperFileName>
```

## Parameters

- **WallpaperFileName**

## Examples

```text
WALL %CurDrv%\outpart\WallPage.JPG
```

## Remarks

- Supportive of environment variables and picture files in various formats.
  This command is applicable to both command line and config file.
